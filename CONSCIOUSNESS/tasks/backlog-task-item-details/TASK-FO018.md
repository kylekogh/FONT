# TASK-FO018: Write FONT-canon enemy names, event text, and card flavor (Kindled theme)

## What

All in-game text grounded in FONT canon — enemy names, event prompts, event outcome text, and any card flavor text not covered in TASK-FO017. Implements FONT-2. This is a writing task, not a build task; it can begin in parallel with implementation work.

## Done when

- **FONT-2 satisfied:** Enemy names, event text, and card flavor are all grounded in FONT canon
- All 8 enemies (TASK-FO010) have Soran names that fit their archetype
- At least 3 event prompts are written in FONT voice with Kindled or Soran lore grounding
- Event outcome text follows FONT sensory register (aphantasia-precise, felt not seen)
- No in-game text uses generic fantasy language, references real-world media, or breaks the FONT vocabulary

## FONT writing rules (reminders)

- Lead with physical sensation (weight, pressure, heat, ache, sound) not visual description
- Enemy names: earned, specific, human-scale — this is a city, not a dungeon
- Event prompts: written as a moment the player is in, not a narrator describing it to them
- "Only the afraid finish what they started" — Soran culture; the city respects those who don't kill unnecessarily
- The Bleed: two lives in one city collide. Event text can hint at this without naming it

## Suggested enemy name register (examples, not locked)

"Limit-Seeker Orin" — a Cultist who went too far  
"The Dimmed" — a Kindled fighter who fought dishonourably and lost their standing  
"Ashfields Runner" — an inner-ring contract fighter  
"Feral Binder" — someone whose bond broke in the dark  
"The Rival" — blank; the player names them at first encounter (named rival mechanic)
