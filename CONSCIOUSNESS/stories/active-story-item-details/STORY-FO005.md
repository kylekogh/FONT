# STORY-FO005: The Kindled — first faction implementation

## Goal

The Kindled as a fully realized playable faction: a cohesive card set (15+) with Kindled identity and flavor text, plus enemy names, event text, and any other in-game text grounded in FONT canon. This is the demo's proof that the FONT world translates to a game system.

## Acceptance criteria (from directive)

- **FONT-1** — At least one fully realized faction with cohesive card set (15+ cards) and flavor text
- **FONT-2** — Enemy names, event text, and card flavor grounded in FONT canon

## Tasks

- **TASK-FO017** — Author 15+ Kindled cards with faction identity and flavor text
- **TASK-FO018** — Write FONT-canon enemy names, event text, and card flavor

## Sequencing

TASK-FO003 (card schema) → TASK-FO017 (Kindled cards need the data model)  
TASK-FO018 can begin independently — it is a writing/design task, not a build task.

## Notes

The Kindled philosophy: combat is the path to unique Spark expression. Cards should feel like they push the fighter toward their next form, not toward domination. Sensory register: heat in the chest, earned ache, the friction of real opponents. Enemy names should feel Soran — lived-in, not fantasy-generic. Reference the established canon: Kuregar (Champion), the Star Path dojo, Ardour as the standing track. Flavor text never names real-world references; FONT speaks in its own vocabulary only.
