# STORY-FO004: Relics and player progression

## Goal

Persistent run modifiers (relics) and a card upgrade path that make each run feel distinct and give players meaningful choices beyond the draft screen.

## Acceptance criteria (from directive)

- **PLR-1** — At least 5 relics implemented that modify run behavior in meaningful ways
- **PLR-2** — Card upgrade mechanic available at rest sites or shops

## Tasks

- **TASK-FO015** — Implement 5 relics with distinct run-modifying effects
- **TASK-FO016** — Build card upgrade mechanic at rest sites

## Sequencing

No hard blockers — relics can be authored in parallel with combat work. TASK-FO016 requires a rest node to exist (TASK-FO012).

## Notes

p2 priority: relics and upgrades are important to the deckbuilder feel but should not block the core loop (card system + combat + run structure). Design relics around the Kindled faction where possible — a relic tied to Ardour or the King's Sight builds FONT cohesion without extra content work.
